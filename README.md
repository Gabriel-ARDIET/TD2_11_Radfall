# Radfall
## Lore
Après une guerre nucléaire mondiale, la terre est dévasté. 
La plupart des espèces ont disparus et les seules survivantes ont subis des mutations étranges à cause des radiations.
Ces espèces survivent dans les sous-terrains, la où le poison des radiations est moins forte.
